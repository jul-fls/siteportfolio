Projet serveur minecraft de Biscuicraft



Ce projet consiste à créer un serveur Minecraft en version 1.12.2.
Ce serveur fonctionne avec des mods (fichiers .jar permettant d'ajouter / modifier le jeu) grâce à l'API Forge.
Il fonctionne également avec des plugins (fichiers .jars permettant d'ajouter des fonctionnalités coté serveur grâce à l'API Spigot.
Normalement ces 2 APIs ne sont pas compatibles mais grâce à ce projet Github (CatServer), les 2 APIs sont réunies dans le même fichier .jar de lancement du serveur.
Ce projet a donc pour buts de :

  • Créer notre propre communauté autour de l'univers de Minecraft en tant que fans de ce jeu
  • Permettre de nous aider financièrement à payer notre serveur qui à son début, était privé (que pour les fondateurs)
  • Me permettre de découvrir le fonctionnement d'une base de données MySQL (les tables, les colonnes, ids, etc..)
  • Me permettre d'apprendre la configuration des mods & plugins Minecraft
  • Me permettre de comprendre l'importance de l'optimisation du developpement sous toutes ses formes, ce qui est loin d'etre le cas pour les serveurs minecraft
  • Faire de nouvelles connaissances et tisser des liens, avec les nouveaux joueurs sur le serveur


En résumé, ce projet, m'a permis de découvrir le projet CatServer que je ne connaissais absolument pas jusque là, le développement en java de plugins et mods, l'administration d'une base de donnée MySQL et tisser des liens, ce qui est sources d'informations et de contacts humains.
