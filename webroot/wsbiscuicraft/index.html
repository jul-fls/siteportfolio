Projet site web de Biscuicraft



Ce projet consiste à créer un site web pour le serveur minecraft Biscuicraft (que nous avons cofondés avec des amis) pour :

  • Avoir une page de téléchargement du launcher de launcher (communément appelé "launcher")
  • Avoir une boutique en ligne pour couvrir nos frais d'hébergements du serveur
  • Réunir toutes les informations principales en un seul et meme endroit
  • Avoir un forum pour que les clients puissent communiquer entre eux
  • Apporter de la visibilité et du référencement au site web
  • Avoir une documentation (Wiki + FAQ)


En résumé, ce projet, m'a permis de modifier moi-même le CMS Mineweb par son côté open-source et documenté, ce qui m'as permis d'approfondir mes connaissances en Php, Css, Html.

Faire la découverte d'un nouveau type de fichier le .ctp, qui est le "CakePhp Template" qui sert aux vues dans CakePHP le framework de ce CMS, qui est globalement rempli de code Php mais pas que.. d'où son extension de fichier si particulière.
